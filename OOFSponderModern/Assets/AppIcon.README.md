# OOFSponder Modern App Icon

## Recommended Version

Use `AppIcon-v2.svg` as the primary modern app icon.

Use `AppIcon-v2-small.svg` for small icon exports such as 16x16, 24x24, 32x32, and tray contexts.

`AppIcon.svg` is the first exploration and is retained only as an archive/reference. It has too many visual elements for a crisp Windows app icon.

## Concept

The v2 icon keeps the legacy OOFSponder `O` recognition while adding product-specific meaning with fewer elements:

- **O ring**: continuity with the existing OOFSponder logo and the out-of-office concept.
- **Clock hand**: schedule-driven automation.
- **Gold away dot**: visible state marker for away / OOF.

## Color Direction

The primary SVG uses the app's modern dark/productivity palette:

- Deep navy background for trust and desktop readability.
- Cyan/teal ring for modern productivity and calm operation.
- Small gold accent for away state and visual recognition.
- White inner marks for small-size legibility.

`AppIcon-Monochrome.svg` is provided for tray, favicon, or high-contrast scenarios.

## Recommended Exports

For Windows `.ico`, export from `AppIcon.svg` at:

- 16x16
- 24x24
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

Keep `AppIcon-Monochrome.svg` available for small monochrome contexts where the full gradient loses detail.
